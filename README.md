# 🌱 Plant Disease Detector

AI-powered plant health analysis using **Transformers.js**. Everything runs in your browser—no server, no API calls, 100% private.

## ✨ Features

- 📸 **Live Camera Capture** - Take photos with your device's camera
- 📁 **Image Upload** - Upload existing plant photos
- 🤖 **AI-Powered Analysis** - PlantVillage model with 38 disease classes
- 📊 **Confidence Scoring** - See how confident the AI is
- 💊 **Treatment Advice** - Get basic recommendations
- 🌙 **Dark Mode** - Auto-adapts to your system theme
- 🎨 **Animated Background** - Beautiful floating leaves
- 🔒 **100% Private** - No images are ever sent to a server
- live link https://glittering-cassata-f5f736.netlify.app/

## 🛠️ Technology Stack

| Component | Technology |
|-----------|------------|
| **Frontend** | HTML5, CSS3, JavaScript (ES6+) |
| **AI Library** | Transformers.js (Hugging Face) |
| **Model** | PlantVillage Dataset (38 classes) |
| **Hosting** | Vercel / GitHub Pages |
